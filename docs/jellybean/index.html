Resistors

Fixed value

Any resistor in E24, E48, E96, or E192 series (lower preferred).

Package Voltage Power
0402 50V 63mW
0603 75V 100mW
0805 150V 125mW

Variable

  • Bourns 3386 9mm square single turn, 0.5 W
  • Bourns 3296 9mm square multi turn, 0.5 W
